



GAME SEQUENCE

1. Player joins field and becomes host
2. Players join field
	IF gametype is active use autosort
	ELSE join unaffiliated

3. Host chooses a gametype;
	GAMETYPE gives out list of allowed team colors. 
	Gametype auto-sorts unaffiliated/disallowed players into list
	
4. Players switch teams if they want to 
	optional check for size matching
	
5. Host presses the START button!
	Match begins
	onPlayerRespawn is called 
	Timer is started
	onGameStart is called 
	
6. Timer runs out
	onGameCancel is called
